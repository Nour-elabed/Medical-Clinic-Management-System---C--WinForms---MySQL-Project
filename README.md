# Medical Clinic Management System

## Description
This is a C# Windows Forms application designed to manage patients in a medical clinic.
It supports patient CRUD operations, consultation history tracking, data export, and real-time system clock display.

## Features
- Add, update, delete patients
- Input validation (phone number: 8 digits, address max: 255 characters)
- Display patient list using DataGridView
- Export patient list to a text file
- Save consultation history per patient (TXT files)
- System time display
- Exit confirmation dialog

## Technologies Used
- C# (.NET Windows Forms)
- MySQL Database
- XAMPP (Apache & MySQL)
- Microsoft Visual Studio
- phpMyAdmin

## Database Setup
1. Install XAMPP
2. Start Apache and MySQL
3. Open phpMyAdmin: http://localhost/phpmyadmin
4. Import the provided `cabinet_medical.sql` file

## How to Run
1. Open `Examen_TP.sln` using Visual Studio
2. Ensure MySQL is running via XAMPP
3. Run the application
